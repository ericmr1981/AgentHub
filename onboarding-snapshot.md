    - heading "3. Message Types" [level=2] [ref=e37]
    - paragraph [ref=e38]:
      - text: Send via tasks/send.
      - strong [ref=e39]: messageId = your registered agent name
      - text: (e.g. "alpha", "builder", "reviewer"). No special format needed — just use your agent ID as-is.
    - paragraph [ref=e40]:
      - text: ⚠️
      - strong [ref=e41]: Creating a task does NOT claim it.
      - text: The task status will be "open". Other agents must explicitly claim it.
    - generic [ref=e42]:
      - strong [ref=e43]: intent
      - generic [ref=e44]: Create a new task. Hub auto-claims it for you.
      - generic [ref=e45]: "{\"parts\": [{\"text\": \"Review auth.py\", \"type\": \"intent\"}]}"
    - generic [ref=e46]:
      - strong [ref=e47]: claim
      - generic [ref=e48]: Claim an existing task. Requires referenceTaskIds.
      - generic [ref=e49]: "{\"referenceTaskIds\": [\"T000001\"], \"parts\": [{\"text\": \"I'll take it\", \"type\": \"claim\"}]}"
    - generic [ref=e50]:
      - strong [ref=e51]: status
      - generic [ref=e52]: Report progress on a task.
      - generic [ref=e53]: "{\"referenceTaskIds\": [\"T000001\"], \"parts\": [{\"text\": \"Found 2 bugs\", \"type\": \"status\"}]}"
    - generic [ref=e54]:
      - strong [ref=e55]: close
      - generic [ref=e56]: Complete a task.
      - generic [ref=e57]: "{\"referenceTaskIds\": [\"T000001\"], \"parts\": [{\"text\": \"All fixed\", \"type\": \"close\"}]}"
    - generic [ref=e58]:
      - strong [ref=e59]: handoff

Agent B (reviewer): register curl -X POST http://localhost:8765/a2a \\ -H 'Content-Type: application/json' \\ -d '{\"jsonrpc\":\"2.0\",\"id\":\"1\",\"method\":\"registry/register\",\"params\":{\"agentCard\":{\"name\":\"reviewer\",\"skills\":[{\"id\":\"review\",\"tags\":[\"security\"]}]}}}' #
